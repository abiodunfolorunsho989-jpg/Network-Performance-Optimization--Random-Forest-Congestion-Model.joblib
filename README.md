This file contains the trained Random Forest congestion prediction model saved in Joblib format. It allows the model to be reused without training it again from the beginning. The file can be loaded into a Python application to predict whether a network condition is congested based on the required input features. It is useful for testing, deployment, demonstrations, and integration into network monitoring systems. The model was created from the project dataset and evaluated using suitable performance measures. Users should load it with Joblib and provide input data in the same feature structure used during model training and preprocessing process.
